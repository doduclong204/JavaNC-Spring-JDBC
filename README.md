## Các bước thực hiện:

### 1. Tải source code về:
```bash
git clone https://github.com/NamVu5404/JavaNC-Spring-JDBC.git
```

### 2. Thiết lập cơ sở dữ liệu:

- Tạo một database mới.
- Import cơ sở dữ liệu từ file `estateadvance.sql` trong source code.

- Chạy ứng dụng.
